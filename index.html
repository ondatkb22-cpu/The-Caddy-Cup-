    PROJECTION MODEL

    • DK pts = scoring_EV + finish_EV
    • scoring_EV = (9.0 + 3.0 × SG/rd) × exp_rounds
    • finish_EV = 30×P(win) + 14×P(T5) + 7×P(T10) + 5×P(T20) + 2×P(T30)
    • exp_rounds = 4×P(cut) + 2×(1−P(cut))
    • σ_scoring = 3.0 × SG_std × √exp_rounds
    • σ_finish = 0.5 × finish_EV + 2.0
    • σ_total = √(σ_scoring² + σ_finish²)

    OPTIMIZER — 8 OBJECTIVES

    • proj — raw expected DK pts
    • ceil_50 — proj + 0.5σ (GPP ceiling)
    • ceil_75 — proj + 0.75σ (aggressive)

    • win_spike — proj + 0.5σ + 15×P(win)
